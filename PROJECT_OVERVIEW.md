# 我爱克隆 - 项目总览

## 🎨 项目亮点

这是一个**专业级**的网站克隆工具，具有以下特色：

### 1. 现代化落地页
- 使用 **Next.js 14 + TypeScript** 构建
- **Tailwind CSS** 提供精美样式
- **Shadcn/ui** 组件库，专业美观
- 响应式设计，支持移动端

### 2. 优雅的用户体验
- 点击按钮弹出克隆对话框，无需跳转页面
- 实时进度显示，用户体验流畅
- 清晰的成功/失败提示
- 自动下载到浏览器下载文件夹

### 3. 强大的克隆功能
- 完整提取所有资源（HTML、CSS、JS、图片等）
- 支持现代框架（React、Vue、Next.js、Angular）
- Selenium 智能渲染动态内容
- 自动识别和提取 UI 组件

## 🎯 核心功能流程

### 用户操作流程
```
访问落地页
    ↓
点击"免费开始使用"
    ↓
弹出克隆对话框
    ↓
输入网址 + 选择渲染模式
    ↓
点击"开始克隆"
    ↓
显示进度条和状态
    ↓
克隆完成，显示成功提示
    ↓
自动下载 ZIP 文件
```

### 后端处理流程
```
接收克隆请求
    ↓
验证 URL
    ↓
选择提取模式
    ├─ 普通模式: HTTP 请求
    └─ 高级模式: Selenium 渲染
    ↓
解析 HTML 结构
    ↓
提取所有资源 URL
    ├─ CSS 文件
    ├─ JavaScript 文件
    ├─ 图片资源
    ├─ 字体文件
    └─ 其他资源
    ↓
并发下载所有资源
    ↓
修复相对路径
    ↓
识别 UI 组件
    ↓
打包成 ZIP 文件
    ↓
返回给前端
```

## 🔧 技术细节

### 前端技术
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件**: Radix UI (通过 Shadcn/ui)
- **图标**: Lucide React
- **状态管理**: React Hooks (useState)

### 后端技术
- **框架**: Flask
- **HTML 解析**: BeautifulSoup4
- **浏览器自动化**: Selenium WebDriver
- **HTTP 请求**: Requests
- **CSS 解析**: cssutils
- **文件处理**: zipfile, tempfile

## 🚀 性能优化

1. **并发下载** - 使用多线程下载资源
2. **智能缓存** - 避免重复下载相同资源
3. **进度模拟** - 前端显示平滑的进度条
4. **异步处理** - 后端异步处理大文件
5. **资源去重** - 自动去除重复的资源 URL

## 🔒 安全特性

1. **本地处理** - 所有数据在本地处理，不上传到服务器
2. **URL 验证** - 验证输入的 URL 格式
3. **超时控制** - 设置请求超时时间
4. **错误处理** - 完善的错误捕获和提示
5. **临时文件清理** - 自动清理临时文件
