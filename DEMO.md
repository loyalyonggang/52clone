**目标网站**: https://example.com

1. 访问 http://localhost:3000

## 💡 使用技巧

### 技巧 1: 选择正确的模式
- **静态网站**: 不勾选高级渲染（更快）
- **React/Vue 网站**: 必须勾选高级渲染
- **不确定**: 勾选高级渲染（更保险）

### 技巧 2: 处理失败
- 检查网址是否正确
- 尝试勾选/取消勾选高级渲染
- 某些网站有反爬虫保护，可能无法克隆

### 技巧 3: 查看结果
- 解压 ZIP 文件
- 打开 index.html
- 查看 README.txt 了解文件结构
- 查看 metadata.json 了解网站信息

## 🎬 视频教程（文字版）

### 第 1 步: 启动项目
```
1. 双击 start-all.bat
2. 等待两个命令行窗口打开
3. 等待服务启动完成（约 10 秒）
4. 打开浏览器访问 localhost:3000
```

### 第 2 步: 开始克隆
```
1. 点击页面上的"免费开始使用"按钮
2. 在弹出的对话框中输入网址
3. 根据网站类型选择是否勾选高级渲染
4. 点击"开始克隆"按钮
```

### 第 3 步: 等待完成
```
1. 观察进度条和状态文本
2. 等待克隆完成（时间取决于网站大小）
3. 看到绿色成功提示
4. 文件自动下载到浏览器下载文件夹
```

### 第 4 步: 查看结果
```
1. 打开浏览器下载文件夹
2. 找到下载的 ZIP 文件
3. 解压到任意位置
4. 双击打开 index.html 查看克隆的网站
```
